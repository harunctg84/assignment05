What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each node is an element, attribute, or piece of content. The DOM allows developers to access, modify, or delete elements dynamically using programming languages like JavaScript. Changes made to the DOM are reflected in the web page instantly. It is essential for creating interactive, dynamic websites.

How do you select an element from the DOM?

To select an element from the DOM, you can use methods like document.getElementById('id') for selecting by ID, document.querySelector('.class' or 'tag') for selecting by class or tag, and document.getElementsByClassName('class') for selecting all elements with a specific class. These methods allow you to access and manipulate elements on a web page.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation allows a parent element to handle events from its child elements using a single listener, taking advantage of event bubbling. It improves performance and handles dynamic content effectively.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using element.setAttribute() or element.getAttribute(), and styles by modifying element.style.property. For example, element.style.backgroundColor = "blue" changes the background color.
